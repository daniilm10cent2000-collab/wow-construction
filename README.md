# TGRAF AI — как запустить

## 1. Откройте папку проекта в терминале

В Cursor или VS Code откройте встроенный терминал (**Terminal → New Terminal** или `` Ctrl+` ``).

Убедитесь, что вы в папке проекта:

```bash
cd c:\Users\danii\OneDrive\Desktop\wow-construction
```

---

## 2. Установите зависимости

Выполните одну команду:

```bash
npm install
```

Дождитесь окончания (появится список пакетов и сообщение вроде `added X packages`). Это нужно сделать один раз (или после добавления новых зависимостей).

---

## 3. Создайте файл с ключом OpenAI

Без ключа чат на сайте не сможет обращаться к AI.

### Где взять ключ

1. Зайдите на **https://platform.openai.com/api-keys**
2. Войдите в аккаунт OpenAI (или зарегистрируйтесь).
3. Нажмите **Create new secret key**, придумайте имя (например, `tgraf-dev`) и скопируйте ключ (он начинается с `sk-...`).  
   Больше его не покажут — сохраните в надёжное место.

### Файл в проекте

В корне проекта (рядом с `package.json`) создайте файл **`.env.local`**.

**Содержимое файла — одна строка** (подставьте свой ключ вместо `sk-...`):

```
OPENAI_API_KEY=sk-ваш_настоящий_ключ_сюда
```

Пример:

```
OPENAI_API_KEY=sk-proj-abc123...
```

- Имя файла именно **`.env.local`** (с точкой в начале).
- Без кавычек, без пробелов вокруг `=`.
- Перезапустите сервер разработки после создания или изменения `.env.local`.

---

## 4. Запустите сайт

В том же терминале (в папке проекта) выполните:

```bash
npm run dev
```

Должно появиться что-то вроде:

```
▲ Next.js 14.2.0
- Local:        http://localhost:3000
```

---

## 5. Откройте сайт в браузере

Перейдите по адресу: **http://localhost:3000**

- Увидите лендинг TGRAF AI.
- В правом нижнем углу — круглая кнопка с иконкой чата. Нажмите её — откроется окно чата.
- Введите сообщение и нажмите «Отправить» или Enter. Ответ придёт от AI (если ключ в `.env.local` указан верно).

---

## Если что-то не работает

| Проблема | Что сделать |
|----------|-------------|
| `npm: command not found` | Установите Node.js с https://nodejs.org (LTS), перезапустите терминал. |
| Ошибки при `npm install` | Выполните снова; при ошибках сети попробуйте позже или другую сеть. |
| Чат пишет «OPENAI_API_KEY не задан» | Создайте `.env.local` в корне проекта с одной строкой `OPENAI_API_KEY=sk-...` и перезапустите `npm run dev`. |
| Сайт не открывается | Проверьте, что в терминале всё ещё запущен `npm run dev` и порт 3000 не занят. |
| Ошибка 503 / «Ошибка OpenAI» | Проверьте ключ на platform.openai.com (действующий, есть баланс/кредиты). |

---

## Дополнительно

- **Остановить сервер:** в терминале нажмите `Ctrl+C`.
- **Сборка для продакшена:** `npm run build`, затем `npm start`.
- **Порт по умолчанию:** 3000. Другой порт: `npm run dev -- -p 3001`.
